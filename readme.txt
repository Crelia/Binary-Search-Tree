================================READ ME=================================================
	HOW TO RUN
		1. type 'g++ *.cpp' into console
		2. type './a.out' to start program
		3. press '1' and type in the file name holding the tree values
		4. Press '2' to print the in-order list of values
		5. Press '3' to remove a value
		6. Press '0' to exit

=========================================================================================

	NOTES
		The code will auto populate the tree values onece the
		filename is entered. It will also Detect any duplicate
		numbers and remove them from the tree.
